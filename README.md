# Quản Lý Bán Vé Tàu
#### .NET Framework 4.7.2 + Crystal Report
### Các chức năng chính:
* **Đăng nhập**
* **Quản lý chuyến tàu** (Thêm, sửa, xóa)
* **Quản lý vé tàu** (Tìm kiếm, xóa vé)
* **In báo cáo**
* **Tìm kiếm vé tàu**
* **Chọn chỗ** (Hiển thị sơ đồ chỗ ngồi và cho phép chọn ghế)
* **Thanh toán**
* **Tìm kiếm vé đã đặt**
---
### Hướng dẫn cài đặt
#### Thiết lập cơ sở dữ liệu
1. **Tạo CSDL:** Mở SQL Server và chạy file SQL
2. **Cập nhật Connection String:** Mở file KETNOI_CSDL.cs và thay đổi chuỗi kết nối

#### Chạy ứng dụng
1. Mở file `.sln` của dự án trong Visual Studio
2. Nhấn nút **Start** để biên dịch và chạy ứng dụng
